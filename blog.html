FAQ

Q1. Difference among var , let & constant?

var: (1) hoisted (always declared at top of scope, global if none) (2) function scope

let: (1) block scope. (2) not redeclarable

const: (1) block scope (2) not reassignable (3) not redeclarable

Q2. Difference between arrow & regular function?

Arrow Function:(1)The arrow (this keyword) function doesn't define its own execution context. (2)If you try to invoke an arrow function prefixed with new keyword, JavaScrip throws an error, (3)no arguments special keyword is defined inside an arrow function.

Regular Function: (1)this value is dynamic.(2)the regular function can easily construct objects. (3)Inside the body of a regular function, arguments is a special array-like object containing the list of arguments with which the function has been invoked.

Q3. Difference among map, forEach , filter , find method?

forEach: Foreach takes a callback function and run that callback function on each element of array one by one.

filter: filter method loop over the array and executes the callback function and check its return value.

find: The find() method returns the value of the first element that passes a test. The find() method executes a function for each array element. The find() method returns undefined if no elements are found. The find() method does not execute the function for empty elements.

map: Map like filter & foreach takes a callback and run it against every element on the array but whats makes it unique is it generate a new array based on your existing array.

Q4. Why do you use template literal string ?

Template literals provide an easy way to interpolate variables and expressions into strings. The method is called string interpolation.
